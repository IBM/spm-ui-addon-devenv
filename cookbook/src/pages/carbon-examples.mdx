### What is provided in the example Carbon components

There are some Carbon component examples already provided in the carbon-addons-devenv package which can be used
as a reference or aid when starting development on new Carbon components. The examples provided have all that is required
to be fully implemented and integrated in a Java Renderer on custom page:

 - The React component
 - Sass
 - Query provider in Apollo Client
 - JavaScript wrapper function

 There is also unit tests, a storybook wrapper, and JavaScript documentation example.

### The list of example components provided
 The examples provided are:
  - PersonFolio
  - Logo

### How can these example components be integrated in a Java renderer
The following code snippets - which don't include the imports - demonstrate how these Carbon component examples can be integrated into a Java Renderer (respectively):

```
public class PersonFolioSampleRenderer extends AbstractViewRenderer {

  /**
   * Constructor.
   */
  public PersonFolioSampleRenderer() {}

  /**
   * */
  @Override
  public void render(final Field field, final DocumentFragment fragment,
    final RendererContext context, final RendererContract contract)
    throws ClientException, DataAccessException, PlugInException {

    final Element carbonWrapperDiv =
      fragment.getOwnerDocument().createElement("div");
    final String carbonWrapperDivID = "CarbonWrapperDiv";
    carbonWrapperDiv.setAttribute("id", carbonWrapperDivID);
    HTMLUtils.appendComment(carbonWrapperDiv, "filler");
    carbonWrapperDiv.setAttribute("class", "spm-custom-component");
    fragment.appendChild(carbonWrapperDiv);
    // label translations override default labels in Carbon component
    // for properly translated labels, you must put them into a `*.properties` file and look them up using the `context.getDataAccessor()` function.
    // final String labels = "{firstnameLabel: â€˜Translated First name', surnameLabel: 'Translated  Surname', ageLabel: 'Translated  Age'}";
    final String labels = "{}";
    final Map<String, String> map = new HashMap<String, String>();

    includeCarbonComponent(context, carbonWrapperDivID, labels, "{}",
      "window.spmcustom.sampleRenderers.PersonFolio.PersonFolioRenderer",
      map);
  }
}
```
