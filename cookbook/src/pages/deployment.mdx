When development is complete on your custom Carbon component, redeploy the component artefacts and place them under source control

1. Run the following command to to copy the generated compiled resources to your web client custom component.

   ```
   npm run deploy
   ```

2. Ensure that you place the component artefacts under source control.
3. Deploy the application EAR file as usual.
